#  Countries Explorer

A responsive web application that displays country data using the REST Countries API.

# Features

-  Fetches country data from REST Countries API
-  Search countries by name
-  Filter countries by region
-  Dynamic country detail page
-  Dark mode with localStorage persistence
-  Fully responsive design

##  Built With

- HTML5
- CSS3
- Vanilla JavaScript
- REST Countries API
